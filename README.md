# Muchen Wen's Personal Academic Website

This is the personal academic website of **Muchen Wen** (温沐晨), built using the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme.

## About

Muchen Wen is a Ph.D. Candidate (expected graduation: June 2026) in the joint program between:
- **City University of Hong Kong** - Department of Information Systems
- **Xi'an Jiaotong University** - School of Management

## Research Focus

Sharing Economy, Empirical Operations Management, Urban Computing

## Key Research Topics

- Spillover effects of bike-sharing on home-sharing (Airbnb) performance
- Impact of bike-sharing on urban crime
- Bike-sharing effects on restaurant patronage
- Spillover effects on taxi markets and driver welfare
- Traffic safety implications of micro-mobility

## Website Structure

- **Home**: Overview and introduction
- **Education**: Academic background and honors
- **Publications**: Research papers and working papers
- **CV**: Complete curriculum vitae

## Deployment

To deploy to GitHub Pages:

```bash
# Install dependencies
bundle install

# Build the site locally
bundle exec jekyll serve

# Deploy to GitHub Pages
# The site will be automatically deployed when pushing to the main branch
```
